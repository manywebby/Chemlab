# Chemlab

Hi!

I'm an intrepid coder who wants to make some code. In this project, I have made a fully-functional digital laboratory suitable for children, students who are studying for their GCSE exam, and even some adults who want to refresh their chemistry skills.

Here are the key functionalities of this project:
- Lab
- Has 60 elements
- 200 compounds
- 803 reactions
- reactivity series built in
- interactive periodic table
- settings
- pH measuring

- Beaker mode - mixing elements and doing reactions
- Distillation mode
- Flame mode - burning metals
- Titrate
- Electrolysis

- You can balance equations
- have guide expriments
- view particles
- quiz mode

- you can also use it as a biology lab as well, as it has many organic substances as well
